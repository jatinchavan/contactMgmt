Implementation Details.

  • Implemented functionality to list and update Contact details.
  • Visually appealing and user friendly display.
  • All the contact details are stored in JSON.
  • New contact can be added by clicking New Contact.
  • Contact can be modified by clicking Edit button.
  • Contact can be deleted by clicking Delete button.
  • On initial loading of application default 3 contacts will be displayed.
  • Implemented Sorting feature (Based on Firstname,Lastname & ID).
  • Firstname and Lastname can comprise the letters Aa to Zz & underscore,hyphen(-).
  • Phone no minimum and maximum limit is 10 and should be numeric only.
  • The user can toggle the contact status to Active/Inactive using Toggle Status button.
